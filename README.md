# Godot2Amiga

Compile Godot games to classic Amiga computers.

## Goals

- Export Godot projects
- Generate optimized C code
- Compile with VBCC/GCC
- Support OCS/ECS/AGA
- RTG support
- Hardware accelerated blitter
- Runs on real Amigas
